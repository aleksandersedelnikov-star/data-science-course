# Files Data Science Cource

<h4>1. Launch Jupyter</h4>
<pre>
  jupyter notebook
</pre>
<h4>2. Create new notebook</h4>
<pre>
  1. Upper right corner tap "New" -> Python3 (ipykernel)
  2. Opened new notebook tab
  3. Rename notebook: File -> Rename -> new_name.ipynb
</pre>
<h4>3. Types of cell</h4>
<pre>
  1. Code - for Python code
  2. Markdown - for formatted text
  3. Raw - for not formatted text
</pre>
<h4>4. Work mode</h4>
<pre>
  1. Edit mode (green frame) - edit context
  2. Command Mode (blue frame) - manage cells
</pre>
<h4>5. Hotkeys</h4>
<pre>
  1. Esc - to Command mode
  2. Enter - to Edit mode
  3. Shift + Enter - execute cell and shift to next cell
  4. Ctrl + Enter - execute cell and stay in it
  5. Command mode: a - add cells up, b - add down, dd - remove
</pre>
<h4>6. Add change files to repo</h4>
<pre>
  git add <./file_name>
  git commit -m "text commit"
  git push -u origin <name_branch>
</pre>
<h4>7. End</h4>
<pre>
  Everything is ready. You can get high!
</pre>
